## Overview

This repository contains the **QField Manual: DWA-M-921 Erosion Survey**, a practical field guide for the **mapping and documentation of soil erosion by water** in accordance with the **DWA guideline DWA-M 921**  
(*Soil erosion by water â€“ Mapping guideline for recording current erosion forms*).

The manual provides:
- An **overview of the DWA-M 921 framework**
- A **standardized approach** to recording current erosion forms
- A **practical implementation** using **QGIS** and **QField** for field surveys

It is intended for use by researchers, technicians, practitioners, and institutions involved in soil erosion monitoring and land management.
